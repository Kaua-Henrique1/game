# assets/sounds/sfx/

Efeitos sonoros (SFX) usados em todos os mini-jogos do projeto.

## Convenção de Nomenclatura

```
{categoria}_{descrição}_{índice:02d}.{extensão}
```

Todos os nomes em `snake_case`, letras minúsculas, sem espaços ou caracteres especiais.

## Arquivos Esperados

| Arquivo | Formato | Mini-jogo | Descrição | Status |
|---------|---------|-----------|-----------|--------|
| `sfx_hit_01.wav` | WAV | RhythmGame | Som de acerto de nota (hit) | [ ] pendente |
| `sfx_miss_01.wav` | WAV | RhythmGame | Som de erro / nota perdida (miss) | [ ] pendente |
| `sfx_match_01.wav` | WAV | DressUpGame | Som de encaixe correto da cabeça no `BodySlot` | [ ] pendente |
| `sfx_victory_01.wav` | WAV | Todos | Som de vitória / conclusão do mini-jogo | [ ] pendente |
| `sfx_jump_01.wav` | WAV | RunnerGame | Som do pulo do `PlayerAvatar` | [ ] pendente |
| `sfx_collision_01.wav` | WAV | RunnerGame | Som de colisão com obstáculo ou inimigo | [ ] pendente |

## Notas

- Formato recomendado: WAV (sem compressão, baixa latência) para efeitos curtos.
- Todos os efeitos devem ter duração curta (< 1 segundo) para não interferir na música de fundo.
- O `sfx_hit_01.wav` é disparado quando o jogador acerta uma nota dentro da janela de ±150ms.
- O `sfx_miss_01.wav` é disparado quando o jogador pressiona a tecla fora da janela ou quando uma nota ultrapassa a `HitZone` sem ser acertada.
- O `sfx_match_01.wav` é disparado junto com o `MatchEffect` (animação de corações/brilhos) no DressUpGame.
