# assets/sprites/dressup/bodies/

Sprites de corpo completo (tronco + pernas combinados) para o mini-jogo **O Encaixe Perfeito** (DressUpGame).

> **Nota:** Esta pasta é destinada a sprites de corpo completo (`body_*`). O projeto atualmente usa pastas separadas para `torsos/` e `legs/`. Caso a implementação migre para sprites de corpo unificado, os arquivos devem ser colocados aqui.

## Convenção de Nomenclatura

```
{categoria}_{descrição}_{índice:02d}.{extensão}
```

Todos os nomes em `snake_case`, letras minúsculas, sem espaços ou caracteres especiais.

## Dimensões

| Categoria | Tamanho do arquivo | Escala de exibição |
|-----------|--------------------|--------------------|
| Corpo do DressUpGame | 128 × 192 px | 1× → 128 × 192 px |

## Estrutura Atual do Projeto

O projeto atualmente usa sprites separados por parte do corpo:
- `assets/sprites/dressup/torsos/` — sprites do tronco (128 × ~96 px)
- `assets/sprites/dressup/legs/` — sprites das pernas (128 × ~96 px)

Se a implementação usar sprites unificados de corpo, os arquivos `body_*` devem ter 128 × 192 px e combinar tronco + pernas em um único arquivo.

## Notas

- A combinação vitoriosa é: `head_kaua.png` + `body_kaua_01.png` (ou `torso_kaua.png` + `legs_kaua.png`).
- Sprites são exibidos em escala 1× (128 × 192 px na tela, sem escalonamento).
