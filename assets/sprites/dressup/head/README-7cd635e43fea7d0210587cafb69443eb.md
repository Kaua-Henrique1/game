# assets/sprites/dressup/head/

> ⚠️ **Esta pasta é um alias/espelho de `assets/sprites/dressup/heads/`.**
>
> Os sprites canônicos de cabeça estão em **`assets/sprites/dressup/heads/`**. Esta pasta (`head/`, no singular) existe por compatibilidade com referências de código que usam o caminho no singular. Não duplique os arquivos — use symlinks ou ajuste o código para apontar para `heads/`.

## Convenção de Nomenclatura

```
{categoria}_{descrição}.{extensão}
```

> Nota: os arquivos de cabeça do DressUpGame não seguem o padrão `_{índice:02d}` pois são identificados diretamente pelo nome do personagem.
