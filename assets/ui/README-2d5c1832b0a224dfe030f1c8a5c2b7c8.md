# assets/ui/

Elementos de interface (UI) usados em todos os mini-jogos: botões, barras de progresso, ícones e efeitos visuais.

## Convenção de Nomenclatura

```
{categoria}_{descrição}_{índice:02d}.{extensão}
```

Todos os nomes em `snake_case`, letras minúsculas, sem espaços ou caracteres especiais.

## Notas

- Os receptores (`ui_receptor_col*`) são os círculos na `HitZone` do RhythmGame que acendem quando a tecla correspondente é pressionada.
- O `ui_effect_match_01.png` é disparado pelo `MatchEffect` no DressUpGame ao encaixar a cabeça no `BodySlot` correto.
- A barra de progresso (`ui_bar_progress_01.png`) exibe `(tempoTotal - tempoDecorrido) / tempoTotal` no RunnerGame.
